Shopping List App

The Challenge

A Thinkful project to improve my knowledge of positioning. Design and code an app that allows users to enter items they need to purchase, check and uncheck items on the list and permanently remove items.

The Work

Designed, wireframe using Balsamiq. Semantic HTML. Built non-interactive version first, adding and removing CSS for desired style. Javascript and jQuery for core functionality. Extras also add a counter for how many items left. And a sliding instruction area.
