Shopping List App

The Challenge

Design and code a web app that allows users to enter items they need to purchase, check and uncheck items on the list and permanently remove items. This Thinkful project improves knowledge of CSS positioning and jQuery basics.

The Work

Wireframed the design using Balsamiq. I loved the idea of building a list against a chalkboard image. But keeping the background static as the list grew proved to be a challenge. Once the JS functionality and basic CSS styles were in place, I used two different background images for a responsive experience. Also added a counter for remaining items and a sliding instruction area. Font is Eraser by David Rakowski.
