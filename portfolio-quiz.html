NYC Quiz App

The Challenge

Another Thinkful project designed to explore Object Oriented Programming with JavaScript. Design and code an interactive quiz app from scratch.

The Work

Designed and built a Q&A quiz all about NYC. This was a great exercise in understanding objects, arrays and MVC. This project was definitely the 'aha moment' understanding objects in JavaScript. Each object includes a question, a possible answer and a cool fact.

The progress bar simulates an NYC subway map, where each correct answer brings the user a stop closer to their destination. This was also a fun way to use pure CSS and JS for animation.
