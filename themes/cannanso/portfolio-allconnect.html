Allconnect.com Rebuild

The Challenge

Create a new theme for the redesigned site. Resolve security issues and improve content organization for administrations.

The Work

Directed the rebuild of allconnect.com. Wrote new PHP templates using Drupal 7 best practices. Eliminated major security vulnerabilities to SQL injections and XSS attacks.

Established a clean, modular structure for markup and Sass/Bootstrap framework. Wrote PHP scripts to export all content into a JSON Schema format, providing a consistent import into any CMS.
