Thinkful Tourism - API Mashup

The Challenge

Final Thinkful assignment to bring together HTML, CSS, JavaScript, jQuery and AJAX in a single project using the API(s) of our choice. Response data required in the JSONP format.

The Work

Designed and built a mock tourism site. Thinkful Tourism features a mashup of the Google Maps API and the Instagram API, allowing users to view information about a remote location before booking a trip.

@googlemaps has a lovely Instagram page. These photos and their locations provided the source data for the "tourist spots". Used AJAX to request JSONP data on each photo and plot their locations as Google Map markers.

Displaying the details of each "tourist spot" required an intuitive UI and several rounds of user feedback from the Thinkful community and mentors. Built a lightbox for the final design.
