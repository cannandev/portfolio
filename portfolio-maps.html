Thinkful Tourism - API Mashup

The Challenge

Final Thinkful assignment bring together HTML, CSS, JavaScript, jQuery and AJAX in a single project - an API Hack.

The Work

Designed and built a mock tourism site. Features a mashup of the Google Maps API and the Instagram API, allowing users to view information about a remote location before booking a trip. Besides building the map and pulling jsonp data on each photo, I relied on feedback from users to build a solid UI. The final iteration involved building a lightbox to display location details.
