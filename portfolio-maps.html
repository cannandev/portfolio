Thinkful Tourism - API Mashup

The Challenge

As a Thinkful student, our last assignment was to bring together HTML, CSS, JavaScript, jQuery and AJAX in a single project using the API(s) of our choice. Response data required in the JSONP format.

The Work

Designed and built a mock tourism site. Thinkful Tourism features a mashup of the Google Maps API and the Instagram API, allowing users to view information about a remote location before booking a trip. Besides building the map and pulling JSONP data on each photo, I relied on several iterations of user feedback to build a solid UI. The final solution involved building a lightbox to display location details.
